# TaskManager-CLI

A simple command-line Task Manager built with Python.  
This project demonstrates working with lists, functions, loops, and user input handling.

## Features

- View all tasks  
- Add a new task  
- Delete a task with confirmation  
- Edit an existing task  
- Mark a task as completed  
- Revert a completed task back to pending  
- Press Enter to exit any submenu  
- Handles invalid inputs gracefully  

## How to Use

1. Clone the repository:

```bash
git clone <YOUR_REPOSITORY_URL>
cd <YOUR_PROJECT_FOLDER>
```

2. Run the Python script:

```bash
python task_manager.py
```

3. Follow the instructions in the terminal to manage your tasks.

## Requirements

- Python 3.8 or higher

## Notes

- Tasks are saved in `tasks.json` automatically.  
- The interface is compact and easy to read.  
- The code is structured to be easily extendable for new features.
